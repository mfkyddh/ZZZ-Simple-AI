# ZZZ白话讲AI

`ZZZ-Simple-AI` 是一个面向零基础职场人士和大学生的 AI 入门知识库项目。

这个项目不追求“大而全”，而是希望用尽量白话、直接、可复述的方式，把 AI 里最重要的概念、使用方法和工作场景讲清楚。

## 项目定位

- 给零基础读者看的 AI 入门知识库
- 强调理解，而不是堆知识点
- 强调个人思考，而不是百科式罗列
- 先做好 Markdown 内容，再逐步生成 HTML

## 我们怎么推进

当前采用“苏格拉底提问式”的方式逐步搭建内容：

1. 先确定一个章节名称。
2. 由用户口述零散想法。
3. 再整理成简化版内容。
4. 经确认后，继续推进下一轮。

## 当前阶段

- 先搭建知识库大纲和首批章节框架
- 后续再沉淀 HTML 生成与维护流程

## 内容格式

后续章节默认采用固定 `5` 块模板：

1. `术语`
2. `一句话解释`
3. `你可以先这样理解`
4. `示例`
5. `你只需要记住`

这样做的目的，是让内容更简洁、更稳定，也更方便后续直接转成网页结构。

## 仓库说明

- `AGENTS.md`：项目级协作规范
- `README.md`：项目简介

后续预计会补充：

- `docs/`：正文内容，采用扁平化结构，使用文件名前缀表示顺序
- `indexes/`：目录与索引
- `changes/`：更新记录
- `site/`：HTML 产物
